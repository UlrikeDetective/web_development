About

To add a bit of whimsy to my site I decided to add some sticky notes! These are being applied to the <aside> HTML element. My approach makes use of some elliptical border-radius and a linear-gradient to fake a “fold” in the paper.

I had initially wanted to do this with just one element, so no wrapper or anything. I got close, but the shadow layer (being applied as a pseudo element at the time) ended up in front of everything else. A standalone shadow layer (and the whole wrapper) can be avoided if you just want to use a box-shadow - but I specifically wanted to adjust elliptical border-radius on this too, to help achieve the look I was after.

A nice noise texture and handwritten-style font really help to bring the idea of a sticky note together, imo. For font I am using Shantell Sans (Links down below).

Problems

Mentioned above, mentioned in the CSS comments, the main problem is that I ended up turning this into a wrapper and having an extra div just for the shadow layer. Not ideal! But it works. It’s also easily omitted and you can still get most of the way there with a box-shadow in CSS.

V2 Update!

I've updated the shadow approach thanks to Cédric Aellen sharing a CodePen that demonstrates how to do it with a pseudo element applied to the wrapper itself (Not the note layer!)

Like this we can do away with the blank <div> being used just for shadow, and it's all a bit more tidy.
