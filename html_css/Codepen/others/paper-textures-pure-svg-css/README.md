# Paper Textures — Pure SVG & CSS

A Pen created on CodePen.

Original URL: [https://codepen.io/ol-ivier/pen/raWowqp](https://codepen.io/ol-ivier/pen/raWowqp).

A complete collection of 41 algorithmic paper textures built entirely with SVG filters and CSS. No images — just code. Click the dropdown to switch between Canson, Kraft, Parchment, Washi, Leather, Canvas, and many more. Each texture uses fractal noise, lighting effects, and color matrices to simulate real paper grain, fibers, and surface imperfections.
