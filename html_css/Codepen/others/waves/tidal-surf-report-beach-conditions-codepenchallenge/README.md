# Tidal — Surf Report & Beach Conditions #CodePenChallenge

A Pen created on CodePen.

Original URL: [https://codepen.io/jerora98/pen/QwGmbQB](https://codepen.io/jerora98/pen/QwGmbQB).

A production-ready surf report website built for the June 2026 CodePen 
"At the Beach" challenge. Features procedurally generated hero waves (summed 
sine octaves with random amplitudes and phases for organic, never-repeating 
crests), a seamlessly scrolling 4-layer parallax wave system, animated SVG 
tide chart with draw-in effect, 7-day forecast with mini wave bars, surf spot 
cards, live condition chips with counter animations, and GSAP ScrollTrigger 
reveals — all ready to drop into a real beach or surf brand website.
