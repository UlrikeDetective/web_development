CSS Gradient Secrets

An exploration of creative uses for CSS gradients.

1. Multi-Stop Linear Gradient Button

This button uses a linear-gradient with multiple color stops to create a distinct, sharp-edged effect rather than a smooth transition. The gradient goes from left to right, creating colored edges with a solid center band.

.multi-btn {
    background: linear-gradient(to right, #ff7e5f, #feb47b 30%, #feb47b 70%, #ff7e5f);
}

2. Conic Gradient Spinner

A conic-gradient is used to create a pie-chart-like effect, perfect for spinners or color wheels. The gradient progresses around a center point. By setting the border-radius to 50%, we make the element a circle.

.spinner {
    background: conic-gradient(from 0deg, #ff0000, #ffff00 90deg, #00ff00 180deg, #0000ff 270deg, #ff0000);
}

3. Repeating Linear Gradient Background

The repeating-linear-gradient creates a pattern of stripes. This example creates diagonal lines by specifying an angle (45deg) and defining a small gradient segment that repeats.

.repeating-bg {
    background: repeating-linear-gradient(45deg, #f0f0f0, #f0f0f0 10px, #e74c3c 10px, #e74c3c 20px);
}

4. Smooth Diagonal Gradient Card

A classic linear-gradient can be used to create a smooth, visually appealing background. By specifying an angle (135deg), the gradient flows diagonally across the element.

Gradient Card

This card has a smooth diagonal gradient background.

.card {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}

5. Animated Gradient Pulse Button

This button uses a large linear-gradient as its background and animates the background-position. This creates a mesmerizing, shimmering effect that's great for call-to-action buttons.

.pulse-btn {
    background: linear-gradient(-45deg, #ee7752, #e73c7e, #23a6d5, #23d5ab);
    background-size: 400% 400%;
    animation: grad-shift 15s ease infinite;
}

@keyframes grad-shift {
    0% { background-position: 0% 50%; }
    50% { background-position: 100% 50%; }
    100% { background-position: 0% 50%; }
}
