A Showcase of Modern CSS Grid Techniques

This page demonstrates different approaches to creating powerful and responsive layouts with CSS Grid, based on the provided style.css.

1. The Modern Standard: repeat(auto-fit, minmax())

This is the workhorse of modern responsive layouts. It creates a grid of columns that automatically wrap and resize based on the available space.

.grid {
    display: grid;
    gap: 1.5rem;
    grid-template-columns: repeat(auto-fit, minmax(min(20rem, 100%), 1fr));
}

2. Asymmetric & Intentional Layouts

For more deliberate and sophisticated designs, you can define explicit tracks to create asymmetric layouts. 

.article-grid {
    display: grid;
    gap: 2rem;
    grid-template-columns: 2fr 1fr 1fr;
}

3. Card Layouts That Actually Work

This approach combines the auto-fitting grid with align-items: stretch to ensure all cards in a row have the same height, creating a clean and uniform look.

.cards {
    display: grid;
    gap: 2rem;
    grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
    align-items: stretch;
}

4. Subgrid (The Game-Changer)

Subgrid allows a nested grid to adopt the track sizing of its parent grid. This is revolutionary for creating consistent alignment within components.

.card-grid {
    display: grid;
    grid-template-columns: repeat(12, 1fr);
    gap: 1.5rem;
}

.card {
    display: grid;
    grid-column: span 4;
    grid-template-columns: subgrid;
    gap: 1rem;
}
